## Installation

Dillinger requires [Node.js](https://nodejs.org/) v16+ and  [Yarn](https://yarnpkg.com/) v1.22+ to run.

Install the dependencies and devDependencies and start the application.

```sh
cd raspodie
yarn
ionic serve --lab
```
